Authentication

Endpoints for user authentication and session management.

Login
POST /en/api/v2/auth/login
Request Body
{
  "email": "shabashvilinika@yahoo.com",
  "password": "123"
}
Get Profile
GET /en/api/v2/auth/profile

Retrieve the authenticated user's profile information.

Logout
POST /en/api/v2/auth/logout

End the current user session.

Category

Manage event categories.

List Categories
GET /en/api/v2/category/list
Query Parameters
Parameter Type Description
company_id integer Filter categories by company ID

Country

List Countries

City

List Cities

Event

Create, manage, and retrieve events.

List Events
GET /en/api/v2/event/{company_id}/list

Retrieve all events for a specific company.

Create Event
POST /en/api/v2/event/{company_id}/create
Request Body
{
  "name": "Event Name",
  "description": "Event Description",
  "category": 3,
  "city": 3,
  "base_price": 211,
  "price_per_person": 100,
  "location": "8"
}
Event Details
GET /en/api/v2/event/details/{event_id}

Get detailed information about a specific event.

List Orders
GET /en/api/v2/event/orders/{event_id}

Retrieve all orders for a specific event.

Update Event
PATCH /en/api/v2/event/update/{event_id}
Request Body
{
  "name": "Updated Name",
  "description": "Updated Description",
  "category": 1,
  "city": 3,
  "base_price": 211,
  "price_per_person": 100,
  "location": "8"
}
Delete Event
DELETE /en/api/v2/event/delete/{event_id}

Permanently delete an event.

Event Images

Upload Image
POST /en/api/v2/event/{company_id}/images
Form Data
Field Type Description
image file Image file to upload
alt_text string Alternative text for the image
is_primary boolean Set as primary image
Update Image
PUT /en/api/v2/event/{company_id}/images/update/{image_id}

Update an existing event image.

Delete Image
DELETE /en/api/v2/event/{company_id}/images/delete/{image_id}

Remove an image from an event.
