üìãOverview

Base URL

https://discount.com.ge

API Versions

  • v1: Authentication endpoints
  • v2: Core business logic (cities, categories, events)
  • v3: Orders and payments

Language Support

The API supports multiple languages through URL prefixes:

  • /en/ - English
  • /ru/ - Russian
